## 一、多表关系

### 1. 一对多（1:N）

在多的一方建立外键，指向一的一方的主键。

### 2. 多对多（M:N）

建立第三张中间表，中间表至少包含两个外键，分别关联双方主键。

### 3. 一对一（1:1）

在任意一方加入外键关联另一方主键，并将该外键设置为唯一（UNIQUE）。

## 二、多表查询

### 1. 内连接（Inner Join）

- 隐式内连接
```sql
SELECT 字段列表
FROM 表1, 表2
WHERE 条件 ...;
```

- 显式内连接
```sql
SELECT 字段列表
FROM 表1 [INNER] JOIN 表2
ON 连接条件 ...;
```

### 2. 外连接（Outer Join）

- 左外连接
```sql
SELECT 字段列表
FROM 表1 LEFT [OUTER] JOIN 表2
ON 连接条件 ...;
```

- 右外连接
```sql
SELECT 字段列表
FROM 表1 RIGHT [OUTER] JOIN 表2
ON 连接条件 ...;
```

### 3. 自连接（Self Join）

```sql
SELECT 字段列表
FROM 表A 别名A
JOIN 表A 别名B
ON 条件 ...;
```

## 三、联合查询（UNION / UNION ALL）

```sql
SELECT 字段列表 FROM 表A ...
UNION [ALL]
SELECT 字段列表 FROM 表B ...;
```

- `UNION ALL`：全部数据直接合并  
- `UNION`：合并后去重
