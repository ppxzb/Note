## 一、触发器类型

### 1. INSERT 型触发器

```text
NEW表示将要或者已经新增的数据
```

### 2. UPDATE 型触发器

```text
OLD表示修改之前的数据，NEW表示将要或已经修改后的数据
```

### 3. DELETE 型触发器

```text
OLD表示将要或者已经删除的数据
```

## 二、语法

### 1. 创建

```sql
 CREATE TRIGGER trigger_name
 BEFORE/AFTER INSERT/UPDATE/DELETE ON table_name FOR EACH ROW
 END;
```

触发器体结构：
```sql
BEGIN
  trigger_stmt;
END;
```

### 2. 查看

```sql
SHOW TRIGGERS;
```

### 3. 删除

```sql
DROP TRIGGER [schema_name.]trigger_name;
```
